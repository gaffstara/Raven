# Java Syntax Basics

Summary

Overview of Java syntax: classes, methods, and types.

Explanation

- Java is statically typed and uses classes for structure.

Example

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, world");
    }
}
```

Notes

- Follow Java naming conventions for readability.
