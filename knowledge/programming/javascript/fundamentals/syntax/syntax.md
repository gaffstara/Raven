# JavaScript Syntax Basics

Summary

Brief overview of JavaScript syntax: functions, variables, and promises.

Explanation

- JavaScript supports both function and arrow function syntax.

Example

```javascript
function greet(name) {
  return `Hello, ${name}`;
}
console.log(greet('world'));
```

Notes

- Prefer strict mode and consistent style.
