# Python Syntax Basics

Summary

A quick overview of Python syntax: indentation, statements, and expressions.

Explanation

- Python uses indentation to define blocks.
- Statements and expressions follow simple rules.

Example

```python
for i in range(3):
    print(i)
```

Notes

- Maintain consistent indentation to avoid SyntaxError.
